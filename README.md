#  BookVerse 

A modern Book Finder web application built using HTML, CSS, and JavaScript with Open Library API.

# Features

- 🔍 Search Books
- 🎤 Voice Search
- ❤️ Favorites
- 📖 Reading List
- ⭐ Book Ratings
- 🌙 Dark / Light Mode
- 📊 Dashboard Statistics
- 🕒 Search History
- 📄 Book Details Modal
- 🔔 Toast Notifications
- 📱 Fully Responsive
- 💾 Local Storage Support

# Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Open Library API




# BookVerse | HTML, CSS, JavaScript, Open Library API

Developed a responsive book discovery platform with voice search, favorites, reading list, ratings, and search history.
Integrated Open Library API for real-time book search and details retrieval.
Implemented LocalStorage for persistent user preferences and dashboard analytics.
Designed modern UI with dark/light mode and mobile-first responsive layout.
